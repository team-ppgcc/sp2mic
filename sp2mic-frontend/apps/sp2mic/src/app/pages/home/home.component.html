SP2MIC - A Tool for Microservice Code Generation from Stored Procedures

A Tool for Microservice Code Generation from Stored Procedures

In general terms, SP2Mic is able to carry out, in a semi-automatic way, the migration of business rules that are implemented in stored procedures to microservices, which aims to help developers and analysts in the evolution of software systems that suffer from obsolescence technical or maintenance necessity.

Migrate stored procedures to microservices, can be considered as a translated process, the stored procedure source code will be read and transformed into another code. In order to have a division of responsibilities and a better organization, the tool was divided in three modules:

  1. Stored procedures loading and processing,
  2. Stored procedures analysis
  3. Microservices code generation.

Functionalities:

  • Stored Procedures - Loading and Processing;
  • Stored Procedures - Analysis
  • Microservices - Create, Read, Update and Delete
  • DTO Class - Read, Update and Delete
  • Microservices - Code Generation
  • Orchestrator - Code Generation
  • Api Gateway - Code Generation
